### Phase 1: 数据模型完善（优先级 P0）

1. **新增表结构**（在 `src/db.ts`）：
   ```sql
   -- 工作区表
   CREATE TABLE workspaces (
     id INTEGER PRIMARY KEY,
     folder TEXT UNIQUE NOT NULL,
     name TEXT NOT NULL,
     owner_user_id INTEGER NOT NULL,
     execution_mode TEXT DEFAULT 'container',
     max_parallel_tasks INTEGER DEFAULT 3,
     created_at INTEGER NOT NULL
   );

   -- 工作区任务表
   CREATE TABLE workspace_tasks (
     id INTEGER PRIMARY KEY,
     workspace_id INTEGER NOT NULL,
     requested_by_user_id INTEGER NOT NULL,
     message TEXT NOT NULL,
     task_type TEXT NOT NULL,  -- quick | simple | complex | background
     status TEXT NOT NULL,      -- queued | running | completed | failed
     priority INTEGER DEFAULT 0,
     queue_position INTEGER,
     started_at INTEGER,
     completed_at INTEGER,
     result TEXT
   );

   -- 邀请码表
   CREATE TABLE workspace_invites (
     id INTEGER PRIMARY KEY,
     workspace_id INTEGER NOT NULL,
     code TEXT UNIQUE NOT NULL,
     expires_at INTEGER NOT NULL,
     used_at INTEGER,
     used_by_user_id INTEGER
   );
   ```

2. **修改 groups 表**：
   ```sql
   ALTER TABLE groups ADD COLUMN workspace_id INTEGER;
   ALTER TABLE groups ADD COLUMN is_shared_workspace BOOLEAN DEFAULT 0;
   ```

### Phase 2: 核心逻辑实现（优先级 P0）

1. **任务分类器**（新建 `src/task-classifier.ts`）：

4. **API 路由**（新建 `src/routes/workspaces.ts`）：
